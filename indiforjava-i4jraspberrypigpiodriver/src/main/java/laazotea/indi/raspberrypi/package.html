Provides the classes necessary to control the Raspberry Pi GPIO ports using the INDI for Java Library. This driver uses de Pi4J library.

Use it with the following command (please note that Java programs that controls Raspberry Pi GPIO ports should be launched with sudo):

> sudo java -jar pathToI4JServerJar/I4JServer.jar -add=pathToRaspberryPiGPIODriver/I4JRaspberryPiGPIODriver.jar

Please Note: Pins numbering is the following (according to the Pi4J library):

@see laazotea.indi.server @see laazotea.indi.driver
