Provides the classes necessary to create INDI Servers and a basic but functional Server.

Usually a INDI Server inherits from INDIServer and implements its abstract methods. In those methods additional filtering can be done to avoid some messages arriving to some Devices or Clients. Thus, it is possible to create filering and security mechanisms.

Please check laazotea.indi.server.INDIBasicServer for a functional example of an INDI Server.

To launch the basic Server just:

> java -jar JavaINDIServer.jar [port]

More info at the INDI for Java Website.
