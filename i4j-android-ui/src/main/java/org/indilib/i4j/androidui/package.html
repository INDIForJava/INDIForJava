Provides the classes to produce a simple Android interface for all INDI Devices, Properties and Elements. This classes are usually instantiated from the corresponding client classes through the method getDefaultUIComponent() in various Client classes.

A simple application example org.indilib.i4j.androidui.INDIAndroidBasicActivity is provided.
